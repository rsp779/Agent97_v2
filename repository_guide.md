# Repository Guide

- `RepositoryFactory.create(data_path)` builds the full repository bundle.
- `customer_repository` exposes customer views and network context.
- `banking_repository` summarizes balances, credit, and exposure.
- `transaction_repository` analyzes spend and velocity.
- `digital_repository` covers devices, sessions, and behavioral signals.
- `fraud_repository` builds the LangGraph investigation package.
- `investigation_repository` handles approvals and case lookups.
- `memory_repository` reads task history and feedback.
- `knowledge_repository` searches operational knowledge.
